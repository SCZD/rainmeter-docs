Clean Rainmeter Uninstall

If you have changed things in the standard Rainmeter setup and want to get back to a "fresh" default installation, here are the steps for a "from scratch" installation.

  • Exit Rainmeter.
  • Download a fresh copy of the installer from Rainmeter.net.
  • Use the "Add and Remove Programs" functionality in Windows to uninstall the Rainmeter program.
  • Completely delete the following folders:

    The Program:
    C:\Program Files\Rainmeter

    Your "Skin" files:
    Windows XP: C:\Documents and Settings\YourName\My Documents\Rainmeter
    Windows Vista, Win7, Win8 & Win10: C:\Users\YourName\My Documents\Rainmeter

    Rainmeter settings and your "Layouts":
    Windows XP: C:\Documents and Settings\YourName\Application Data\Rainmeter
    Windows Vista, Win7, Win8 & Win10: C:\Users\YourName\Appdata\Roaming\Rainmeter

  • Right-click and delete the following Windows Start menu items:

    Rainmeter icon
    Rainmeter in the Startup folder

  • Re-install Rainmeter using the downloaded installer.
